隱私權政策

最後更新日期：2026-03-12

本頁說明「成績分析平台」（以下簡稱本平台）是如何蒐集、使用、保存與保護您在使用服務時提供或產生的資料。

1. 適用範圍

本說明適用於本平台網站與相關 API 服務，不適用於第三方網站或外部服務。

2. 我們處理的資料類型

  • 登入資料：您輸入的學校系統帳號與密碼，僅用於向學校系統發起登入請求。
  • 驗證資料：Cloudflare Turnstile 驗證 token，用於機器人防護。
  • 查詢結果：學校系統回傳的成績資料。
  • 系統運作資料：必要的伺服器日誌與錯誤資訊，用於除錯與維運。

3. 資料使用目的

  • 完成登入、取得可查詢考次、查詢成績並顯示於儀表板。
  • 在您主動要求時建立短期分享連結。
  • 維持服務安全性、可用性與故障排查。

4. 資料儲存位置與保留時間

  • 瀏覽器端：成績資料會儲存在您的瀏覽器 localStorage。
  • 伺服器 Session：登入後必要狀態由 Redis Session 保存，有效期為 1 天。
  • 分享資料：建立分享連結後，資料會寫入 Redis， 2 小時自動過期。

本平台不提供長期成績資料庫，不會建立永久的個人成績檔案。

5. 第三方服務

  • 學校系統 API：用於登入與成績查詢。
  • Cloudflare Turnstile：用於人機驗證與濫用防護。

上述第三方服務可能依其政策處理必要技術資訊，請參閱其官方隱私政策。

6. Cookie

本平台使用 Session Cookie 維持登入狀態。

7. 資料安全措施

  • 使用 HTTPS/TLS 傳輸。
  • 限制 API 請求大小與靜態檔案類型。
  • 針對外部請求採用 timeout、retry 與憑證驗證策略。
  • 分享連結採隨機 ID，並設定自動到期。

8. 您可採取的控制方式

  • 可隨時登出以清除伺服器 Session。
  • 可在瀏覽器清除本站資料以移除 localStorage 內容。
  • 如不希望資料被他人檢視，請勿建立分享連結，或等待分享連結自動過期。

9. 說明更新

本平台可能因功能或法規調整而更新本說明。更新後版本將公布於本頁，並以頁面上的「最後更新日期」為準。

10. 聯絡方式

若您對本說明有疑問，請透過 contact@clhs.dev 與我們聯繫。
